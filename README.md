# ielts-typing-learner

一个简单的雅思自定义生词拼写练习器。学生可以按阅读文章、真题、课程或日期创建多个词库，把英文生词和中文意思手动输入进去，然后针对当前词库随机练习英文拼写。

## 使用场景

- 雅思阅读、听力、写作备考时整理当天生词
- 按“剑雅18 Test 1 Passage 1”这类名称管理不同文章的词库
- 课后复习容易拼错的单词
- 用中文意思反向训练英文拼写
- 在本机浏览器中保存个人词库，随时继续练习

## 功能列表

- 创建多个词库
- 在左侧切换当前词库
- 批量录入生词，格式为：`英文单词 - 中文意思`
- 自动解析多行输入内容，并添加到当前选中的词库
- 将词库保存到 `localStorage`
- 显示当前词库列表
- 支持编辑单个单词
- 支持删除单个单词
- 支持确认后清空整个词库
- 只从当前词库随机抽取单词进行练习
- 显示中文意思和首字母提示
- 支持按 Enter 或点击按钮提交答案
- 自动判断拼写是否正确
- 统计正确次数、错误次数、已练习次数

## 如何运行

这个项目只使用 HTML、CSS 和原生 JavaScript，不需要安装任何工具。

1. 打开项目文件夹。
2. 双击 `index.html`。
3. 浏览器打开后即可使用。

也可以在 VS Code 中安装 Live Server 插件，然后右键 `index.html`，选择用 Live Server 打开。

## 输入示例

```text
evidence - 证据；根据
sustainable - 可持续的
estimate - 估计
```
